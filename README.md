# CodeLeap Network - Frontend Challenge

[Português](#português) | [English](#english)

<a name="português"></a>
## 🇧🇷 Português

Repositório para o desafio de frontend da CodeLeap, desenvolvido com React, TypeScript e uma interface simulando uma rede social.

### Tecnologias Utilizadas

- **Core**: React 18 & Vite
- **Linguagem**: TypeScript
- **Estilização**: CSS Puro (Custom Properties)
- **Gerenciamento de Estado**: Context API
- **Persistência**: LocalStorage (Mock API)

### Funcionalidades Implementadas

- **Gestão de Posts**: Criação, edição e exclusão de publicações.
- **Interface Responsiva**: Design adaptável para dispositivos móveis e desktop.
- **Infinite Scroll**: Carregamento automático de posts ao rolar a página.
- **Busca**: Filtro de posts por título, conteúdo ou usuário.
- **Tema Escuro**: Suporte a modo escuro integrado.
- **Sistema de Feedback**: Toasts para notificações de sucesso ou erro.

<a name="english"></a>
## 🇺🇸 English

Repository for the CodeLeap frontend challenge, built with React, TypeScript, and a social network-like interface.

### Tech Stack

- **Core**: React 18 & Vite
- **Language**: TypeScript
- **Styling**: Pure CSS (Custom Properties)
- **State Management**: Context API
- **Persistence**: LocalStorage (Mock API)

### Implemented Features

- **Post Management**: Create, edit, and delete posts.
- **Responsive Interface**: Design adapted for mobile and desktop devices.
- **Infinite Scroll**: Auto-loading posts as you scroll.
- **Search**: Filter posts by title, content, or user.
- **Dark Mode**: Built-in support for dark theme.
- **Feedback System**: Toast notifications for success or error feedback.

## Como Executar / How to Run

1. Clone o repositório / Clone the repository:
   ```bash
   git clone <url-do-repositorio>
   ```

2. Instale as dependências / Install dependencies:
   ```bash
   npm install
   ```

3. Inicie o servidor / Start the server:
   ```bash
   npm run dev
   ```

Desenvolvido como parte do processo seletivo da CodeLeap. / Developed as part of the CodeLeap recruitment process.
